NoSQL 跟 SQL 的差別在哪裡？

tags: 資料庫

NoSQL是Not Only SQL的縮寫，與 SQL（關聯式資料庫）有以下差別：

  1. 沒有 schema，通常以 JSON、XML 或 CSV 等檔案類型儲存
  2. 用 key-value 來存資料
  3. 不支援 JOIN
  4. 通常用來存一些結構不固定的資料（log 之類的）
