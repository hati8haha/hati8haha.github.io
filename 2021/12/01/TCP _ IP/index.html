tags: web 基礎

網路的層級

OSI 七層

TCP / IP 四層

IP

IPv4 IPv6 不同版本的 IP

不同類型 IP

固定 IP
一台電腦一個 IP，server 會用固定 IP
浮動 IP
每次連線時 IP 不同
虛擬 IP
內網內可互相連接，外網無法連至內網 IP

PORT

區別同一台電腦上的不同服務
HTTP：80
HTTPS：443
FTP：21

表格

SOAP Simple Object Acess Protocol

透過 xml 進行資料交換

不是 SOAP 的其他 HTTP API

RESTful

它不是一個協定，是一種風格

符合語意，直覺的方式
